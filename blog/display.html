What does it mean to display inline vs inline blocks?

TECHNICAL


written by Justin Cole

Nov 17 2020


Every HTML element has a default display value, depending on what type of element it is. The two default display values are: block and inline.


Display Inline


An inline element does not start on a new line and it only takes up as much width as necessary. Some common inline elements are:


  • <a>
  • <button>
  • <img>

If an element is given the display attribute of inline or is inline by default, the height and width properties will have no effect.


Note: An inline element cannot contain a block-level element!



Display Block


A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). Some common block elements are:


  • <div>
  • <header>
  • <nav>
